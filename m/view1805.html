L245NB螺旋管和L245螺旋管有什么不同—天津万盛
L245螺旋管就是一种低压流体卷制管，是螺旋管 线管的一种代表性材质。其性能和20#无缝管差不多广泛用于：用于石油、天然气工业中的气、水、油输送 等领域。

L245螺旋管是把抽出地面的油,汽,水,通过管线管输送到石油和天然气工业企业.管线管包括无缝管和焊接钢管,其管端有平端,带螺纹端和承口端；其连接方式为端头焊接,接箍连接,承插连接等.

化学成分: C：0.26 Mn：1.15 S ： 不大于0.030 P：不大于0.030
力学性能： 屈服强度：245 MPa 抗拉强度：415 MPa
标准：　　执行标准： API SPEC 5L GB/T9711.1 GB/T9711.2

于石油管线管的一种。比L245材质要好，L245NB中N的表示是L245的钢材进了形变火。 B的意思是B级钢管。所以L245NB表示的是在L245的基础上进行了形变正火。

　　L245NB石油管线管执行标准：GB9711.2-1999
